---

Local Layer 是一款浏览器扩展，让你可以直接在网页上点击文本进行翻译，并将结果保存下来。当相同内容再次出现时，系统会自动使用已经保存的翻译结果，无需再次翻译。

随着使用时间增加，Local Layer 会逐渐形成属于你的个人翻译库，让网页阅读越来越轻松。

# ✨ 为什么使用 Local Layer

在浏览外语网页时，人们常常需要反复翻译相同的内容，例如：

* 技术文档中的术语
* 常见错误信息
* API 描述
* 教程中的固定表达

传统方式需要不断重复翻译，而 Local Layer 让这些内容只需要翻译一次。

**一次翻译，之后自动复用。**

---

# 🚀 核心功能

## 点击翻译

开启翻译模式后，只需点击网页中的文本即可查看翻译结果，无需复制粘贴或离开当前页面。

---

## 一次翻译，永久使用

翻译结果会被保存下来。
当同样的文本再次出现在网页中时，系统会自动使用已经存在的翻译结果。

---

## 自动积累个人翻译库

每一次翻译都会成为你个人翻译库的一部分。
随着时间推移，系统能够复用的内容会越来越多。

---

## 悬浮显示翻译结果

翻译内容会以悬浮窗口的方式直接显示在网页上，不打断阅读流程，也不会影响原网页内容。

---

## 翻译结果导入与导出

Local Layer 支持将已经保存的翻译结果 **导出为文件**，方便备份或迁移到其他设备。

同时也支持 **导入翻译数据**，可以快速恢复历史翻译记录。

---

## 翻译库分享

用户可以将自己的翻译结果导出并分享给他人使用。
通过导入这些数据，其他人也可以直接复用已有翻译，从而减少重复翻译的工作。

这使得翻译内容不仅可以个人积累，也可以在团队或社区中共享。

---

## 适用于任何网站

Local Layer 可以在绝大多数网页中使用，例如：

* 技术文档
* 开发者社区
* 在线教程
* 博客文章
* 软件后台界面

无论在哪里阅读外语内容，都可以直接使用。

---

# 🖱 使用方式

1. 打开扩展并开启翻译模式
2. 在网页中点击需要翻译的文本
3. 查看翻译结果
4. 保存翻译内容
5. 当相同文本再次出现时自动复用

---

# 📈 使用效果

使用 Local Layer 一段时间后，你会发现：

* 越来越多内容不需要再翻译

翻译的次数越多，系统就越有用。
